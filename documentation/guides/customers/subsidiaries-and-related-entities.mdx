---
title: 'Subsidiaries & related entities'
description: 'Create hierarchy between entities and associate child entities'
---

It is possible to create hierarchy between entities and associate 'child' entities with a parent account . This provides you with high level of customization either to issue an invoice to each entity / subsidiary separately or aggregate the charges to the main parent account.
<Steps>
  <Step title="First Step">
  Go to customers and select the customer account (parent) you wish to edit
  </Step>
  <Step title="Second Step">
  go to --> Settings and scroll down to find the related entities section
  </Step>
  <Step title="Third Step">
    click the "Add" icon --> select the customer you wish to add as related entity. It is possible to add unlimited number of entities.
  </Step>
</Steps>

once done make sure to "save".
