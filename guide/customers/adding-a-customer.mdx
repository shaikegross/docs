# Adding a Customers

 -   Customers can be added in one of three ways:
	 - Imported via CSV file
	 - Salesforce or Hubspot integration
	 - Adding manually - as described below

-   To initiate the customer creation process, start by selecting "Customer" on the main menu and then click the "+" button.
    
-   Next, add all required information, including Customer Name, Address, and Contact details.

- Once completed, simply click the "save" button to finalize the process.
